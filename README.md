# tokamak

> A tokamak confines plasma. This one confines tokens.

A minimal LLM inference engine built from scratch in PyTorch — KV caching, paged
attention, continuous batching, and speculative decoding — benchmarked honestly
against [vLLM](https://github.com/vllm-project/vllm).

**Status: under active development.** See the [roadmap](docs/ROADMAP.md) for the
milestone plan and current progress.
